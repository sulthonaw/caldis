## Latar Belakang

Disleksia merupakan salah satu **gangguan belajar dengan prevalensi tinggi**, namun masih **kurang terdeteksi dan dipahami di Indonesia**. Berbagai sumber menunjukkan bahwa sekitar **10–15% anak usia sekolah mengalami disleksia**, atau setara dengan **4–5 anak dalam satu kelas berisi 40 siswa**. Estimasi lain menyebutkan sekitar **5 juta dari 50 juta anak sekolah di Indonesia mengalami disleksia**, dengan potensi **2 juta kasus baru setiap tahun**.

Namun, data lokal menunjukkan **ketimpangan dalam deteksi**. Di sekolah inklusi Surabaya, prevalensi yang tercatat hanya **0,96%**, yang mengindikasikan **rendahnya kemampuan identifikasi**. Sebaliknya, studi di Bali menunjukkan angka yang jauh lebih tinggi, yakni **20–40% siswa SD terindikasi mengalami disleksia visual**. Perbedaan ini menegaskan bahwa permasalahan utama bukan hanya pada jumlah kasus, tetapi juga pada **kurangnya awareness, deteksi dini, dan metode pembelajaran yang sesuai**.

Di sisi lain, penanganan disleksia membutuhkan **pendekatan khusus** seperti **metode fonik, multisensori, serta pembelajaran yang dipersonalisasi**. Sayangnya, pendekatan ini **belum banyak diimplementasikan secara luas**, terutama pada jenjang **PAUD dan TK yang merupakan fase krusial perkembangan literasi dasar**.

## Problem Statement

- Rendahnya awareness masyarakat dan tenaga pendidik terhadap disleksia  
- Minimnya deteksi dini pada anak usia PAUD–TK  
- Kurangnya media pembelajaran yang sesuai dengan kebutuhan anak disleksia  
- Metode pembelajaran konvensional belum mengakomodasi pendekatan multisensori dan interaktif  
- Akses terhadap edukasi dan penanganan disleksia masih terbatas dan tidak merata  

---

## Tujuan

- Meningkatkan pemahaman masyarakat tentang disleksia  
- Menyediakan media edukasi yang mudah diakses bagi orang tua dan guru  
- Membantu anak usia dini dengan disleksia dalam belajar calistung  
- Menghadirkan metode pembelajaran yang interaktif dan berbasis multisensori  

---

## Solusi

Solusi yang dikembangkan adalah sebuah website edukatif yang berfungsi sebagai:

### 1. Media Edukasi Disleksia
Menyediakan informasi dasar mengenai disleksia, ciri-ciri, serta cara penanganannya bagi orang tua dan guru.

### 2. Platform Pembelajaran Interaktif (Calistung)
Dirancang khusus untuk anak PAUD–TK dengan pendekatan visual dan multisensori:
- **Menghitung** → menggunakan visual objek (misalnya buah)  
- **Menulis** → aktivitas menarik garis mengikuti bentuk huruf  
- **Membaca** → menebak kata berdasarkan gambar  

### 3. Pendekatan Berbasis Riset
- Metode ejaan (spelling method)  
- Pendekatan multisensori (visual, audio, interaksi)  
- Instruksi fonik eksplisit  
- Pembelajaran adaptif sederhana  

---

## Manfaat

- **Bagi Anak:** Membantu belajar membaca, menulis, dan berhitung dengan lebih mudah dan menyenangkan  
- **Bagi Orang Tua:** Memberikan panduan praktis dalam mendampingi anak dengan disleksia  
- **Bagi Guru:** Menjadi media pendukung pembelajaran inklusif  
- **Bagi Pendidikan:** Mendorong peningkatan awareness dan deteksi dini disleksia di Indonesia

## Instalasi

### Prasyarat

Pastikan sudah menginstall:

* **Node.js**
* **npm** (biasanya sudah termasuk dalam Node.js)

Cek versi dengan:

```bash
node -v
npm -v
```

---

## Langkah Instalasi

1. **Clone repository**

```bash
git clone https://github.com/sulthonaw/caldis.git
```

2. **Masuk ke folder project**

```bash
cd caldis
```

3. **Install dependencies**

```bash
npm install
```

4. **Jalankan project**

```bash
npm run dev
```

---

Project akan berjalan di:

```
http://localhost:3000
```
